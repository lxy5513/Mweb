cap = cv2.VideoCapture(video_name)

## write video by frame
```
ret, frame = vs.read() 
H,W = frame.shape[:2]
output_fps = vs.get(cv2.CAP_PROP_FPS)
fourcc = cv2.VideoWriter_fourcc(*'mp4v')
# ⚠️读取时是（高、宽、3）， 写入是设置是（宽、高）
out = cv2.VideoWriter(video_output,fourcc, output_fps, (W,H))
```

cap = cv2.VideoCapture(video_name)
