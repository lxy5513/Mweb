# neovim 

neovim和vim是同级并列的存在 

安装方式很简单
```
1 下载：
wget https://github.com/neovim/neovim/releases/download/v0.4.3/nvim-linux64.tar.gz -O nvim-linux64.tar.gz 

2 解压 
tar -xvf nvim-linux64.tar.gz

3 将bin路径加入bash_profile 

4 alias vim='nvim'
```

<br>

安装nvim的python3支持 

python3 -m pip install --user --upgrade pynvim
